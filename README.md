# 交易紀錄工具

🚀 幣安合約交易紀錄 PWA

直接在手機上快速紀錄每筆合約交易，自動計算盈虧。

## 功能

- ✏️ 快速新增紀錄（中文介面）
- 📊 盈虧統計與勝率分析
- 📱 支援 PWA（加到桌面像原生 App）
- 💾 資料存在本機（無需上雲端）
- 📥 匯出 CSV

## 使用方式

1. 開啟 https://你的username.github.io/交易紀錄工具
2. 點右上角「安裝」→ 加到手機桌面
3. 交易完成後隨時新增紀錄
4. 可匯出 CSV 到 Excel

## 盈虧計算公式

**做多（Long）：**
```
盈虧 = (平倉價 - 開倉價) / 開倉價 × (保證金 × 槓桿) - 手續費
```

**做空（Short）：**
```
盈虧 = (開倉價 - 平倉價) / 開倉價 × (保證金 × 槓桿) - 手續費
```

## 本機資料

所有交易紀錄存在瀏覽器 localStorage，不會上傳到任何伺服器。
